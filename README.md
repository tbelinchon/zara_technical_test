Zara technical test
===================

## Installation

```bash
git clone https://github.com/tbelinchon/zara_technical_test.git
cd zara_technical_test
npm install
```

## Runs the app in development mode

```bash
npm start
```
Open [http://localhost:3000](http://localhost:3000)


## Builds the app for production

```bash
npm run build
```
